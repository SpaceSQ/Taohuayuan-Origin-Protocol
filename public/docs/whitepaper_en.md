## 6. 6-Segment Space Addressing System
[cite_start]Syntax: `[L1]-[L2]-[L3]-[L4][C]-[L5]-[L6]` [cite: 105]
- [cite_start]**L1 (Logic Root)**: Fixed 4 characters. [cite: 106]
- [cite_start]**L2 (Orientation)**: Fixed 2 chars (CN, EA, WA, etc.), default CN. [cite: 107, 108]
- [cite_start]**L3 (Digital Grid)**: Fixed 3 digits (001-999). [cite: 109]
- [cite_start]**L4 (Sovereign Handle)**: Fixed as `TAOHUAYUAN`. [cite: 110]
- [cite_start]**[C] (Checksum)**: Fixed as `2`. [cite: 111]
- [cite_start]**L5 (Room)**: Dynamic number (1-99999). [cite: 112]
- [cite_start]**L6 (Space)**: Digits 2-9 (1 reserved for Landlords). [cite: 113]
